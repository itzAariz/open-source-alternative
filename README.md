# Open Source Alternatives

A curated list of professional **Open Source Alternatives** to proprietary software and services from "Big Tech" companies. This repository aims to provide high-quality, privacy-respecting, and community-driven options for everyday digital needs.

## Search Engines
| Proprietary | Open Source / Privacy Alternative | Description |
| :--- | :--- | :--- |
| **Google Search** | [Brave Search](https://search.brave.com/ ) | A privacy-respecting. |
| **Bing** | [DuckDuckGo](https://duckduckgo.com/ ) | Privacy-focused search (partially proprietary but uses open source components). |
| **Baidu** | [Whoogle](https://github.com/benbusby/whoogle-search ) | Self-hosted Google search results without ads or tracking. |

## Web Browsers
| Proprietary | Open Source Alternative | Description |
| :--- | :--- | :--- |
| **Edge** | [Firefox](https://www.mozilla.org/firefox/ ) | The gold standard for open-source web browsing. |
| **Safari** | [Brave](https://brave.com/ ) | Chromium-based with built-in ad and tracker blocking. |
| **Chrome** | [Helium](https://helium.computer/ ) | A custom version of Chromium focused on privacy and security. |

4. Submit a pull request.
